## 1. Méthodes de révision efficaces

### La récupération active (active recall)

Le cerveau retient mieux quand il PRODUIT l'information plutôt que quand il la RELIT.

**En pratique :**
- Lire le cours une fois, puis fermer le support et essayer de restituer de mémoire
- Se poser des questions : "Quels sont les 3 principes de X ?" AVANT de vérifier
- Utiliser des flashcards (Anki, papier, ou simplement un cahier plié en deux)
- Expliquer le concept à quelqu'un (ou à voix haute, seul)

**Ce qui ne marche PAS (ou mal) :**
- Relire le cours 5 fois en surlignant → illusion de maîtrise
- Recopier le cours mot pour mot → travail mécanique, pas cognitif
- Regarder des corrections sans avoir essayé l'exercice d'abord

### La répétition espacée

Plus efficace de réviser 4 fois 20 minutes sur 4 jours que 1 fois 80 minutes.

Planning type pour un concept :
- Jour 1 : apprendre
- Jour 2 : réviser (10 min)
- Jour 4 : réviser (5 min)
- Jour 7 : réviser (5 min)
- Jour 14 : test final

Outil gratuit : Anki (flashcards avec algorithme de répétition espacée intégré).

### L'entrelacement (interleaving)

Ne pas réviser un seul sujet pendant 3 heures. Alterner :
- 45 min de compta → 45 min de droit → 45 min de compta
- Mélanger les types d'exercices dans une même session

Le cerveau apprend mieux quand il doit distinguer et catégoriser.

### La technique Feynman

1. Écrire le concept en haut d'une feuille
2. L'expliquer en termes simples, comme à un collégien
3. Identifier les points où l'explication bloque
4. Retourner au cours pour ces points précis
5. Simplifier encore

Si tu ne peux pas l'expliquer simplement, tu ne l'as pas compris.

## 2. Planifier ses révisions

### Planning de révision — méthode

**Étape 1 — Inventaire.** Lister toutes les matières et chapitres à réviser.
Estimer le temps nécessaire par chapitre (léger / moyen / lourd).

**Étape 2 — Prioriser.**
| Priorité | Critère |
|----------|---------|
| Haute | Fort coefficient + faible maîtrise |
| Moyenne | Fort coefficient + bonne maîtrise OU faible coefficient + faible maîtrise |
| Basse | Faible coefficient + bonne maîtrise |

**Étape 3 — Répartir dans le temps.**

Exemple : 2 semaines avant les partiels, 5 matières

```
Semaine 1 (fondations) :
  Lun : Matière A (chapitre lourd) + Matière B (chapitre léger)
  Mar : Matière C (chapitre lourd)
  Mer : Matière A (chapitre moyen) + Matière D
  Jeu : Matière B (chapitre lourd) + Matière E
  Ven : Matière C (chapitre moyen) + exercices Matière A
  Sam : Exercices B + C
  Dim : Repos ou révision légère des points faibles

Semaine 2 (consolidation) :
  Lun : Annales Matière A
  Mar : Annales Matière B + C
  Mer : Points faibles identifiés + fiches
  Jeu : Annales Matière D + E
  Ven : Dernière révision rapide (fiches uniquement)
  Sam : Examen blanc chronométré
  Dim : Repos
```

### Le planning Pomodoro pour les révisions

- 25 min de travail concentré
- 5 min de pause (se lever, boire, regarder par la fenêtre — PAS le téléphone)
- Après 4 cycles : pause de 15-20 min
- Objectif : 8-12 pomodoros par jour de révision intense

## 3. Fiches de révision — formats par matière

### Matières conceptuelles (philo, SES, sociologie, management...)

```
FICHE — [Matière] — [Thème / Chapitre]

AUTEURS CLÉS
• [Auteur 1] ([date]) : [théorie en 1 phrase] — Œuvre : [titre]
• [Auteur 2] ([date]) : [théorie en 1 phrase] — Œuvre : [titre]

CONCEPTS
• [Concept 1] : [définition courte]
  → Lié à : [autre concept]
  → Exemple : [illustration concrète]

DÉBATS / OPPOSITIONS
• [Auteur A] vs [Auteur B] sur [question] :
  A dit... / B dit... / La nuance c'est...

PLAN TYPE (si le thème tombe en dissert)
I. [Axe 1]
II. [Axe 2]
III. [Axe 3]

CITATIONS UTILES (max 3)
• "[citation]" — [Auteur], [Œuvre], [date]
```

### Matières techniques (compta, droit fiscal, finance, maths...)

```
FICHE — [Matière] — [Thème]

FORMULES / RÈGLES
• [Formule 1] : [nom] = [expression]
  → Quand l'utiliser : [contexte]
  → Piège : [erreur fréquente]

• [Règle juridique / comptable] : Article [X] du [Code]
  → Principe : [en 1 phrase]
  → Exception : [s'il y en a]

MÉTHODE TYPE
1. Identifier [quoi]
2. Calculer / Appliquer [formule/règle]
3. Vérifier [comment]
4. Conclure [avec quelle formulation]

EXERCICE RÉSOLU
Énoncé : [cas court]
Étape 1 : [calcul / raisonnement]
Étape 2 : [suite]
Résultat : [réponse]

ERREURS FRÉQUENTES
• Confondre [X] et [Y] → la différence : ...
• Oublier de [étape critique]
• Appliquer [règle] alors que [exception s'applique]
```

### Matières à mémorisation (histoire, géographie, biologie...)

```
FICHE — [Matière] — [Thème]

CHRONOLOGIE / REPÈRES
• [Date 1] : [événement] → importance : [pourquoi ça compte]
• [Date 2] : [événement] → lien avec : [événement précédent]

ACTEURS CLÉS
• [Personne / pays / institution] : rôle dans [contexte]

CAUSES → ÉVÉNEMENT → CONSÉQUENCES
[Schéma causal en 3 étapes]

VOCABULAIRE SPÉCIFIQUE
• [Terme 1] : [définition]
• [Terme 2] : [définition]

CARTE MENTALE (description textuelle)
[Concept central]
├── Branche 1 : [sous-thème]
│   ├── [détail]
│   └── [détail]
├── Branche 2 : [sous-thème]
└── Branche 3 : [sous-thème]
```

## 4. Préparation par type d'épreuve

### QCM

- Lire TOUTES les propositions avant de répondre
- Attention aux doubles négations ("lequel n'est PAS incorrect")
- En cas de doute, éliminer les réponses manifestement fausses d'abord
- Vérifier s'il y a des points négatifs (si oui, ne pas répondre au hasard)
- Réviser les DÉTAILS : les QCM testent la précision, pas la compréhension globale

### Cas pratique (droit, compta, gestion)

Structure de réponse :
1. **Qualifier juridiquement / comptablement** les faits
2. **Identifier la règle applicable** (article, norme, principe)
3. **Appliquer** la règle aux faits
4. **Conclure** (réponse à la question posée)

Piège : ne pas se lancer dans les calculs ou le raisonnement sans avoir d'abord
qualifié la situation. La qualification, c'est 50% de la note.

### Dissertation (en examen)

- Lire le sujet 3 fois
- Brouillon : 30-40% du temps total
- Rédaction : 50-60%
- Relecture : 10%
- Si le temps manque : rédiger intro + conclusion en entier,
  rédiger les idées directrices de chaque partie sans développer

### Oral / Grand Oral

- Préparer des notes structurées (plan + mots-clés), pas un texte rédigé
- S'entraîner devant un miroir ou en s'enregistrant
- Parler lentement (on parle toujours plus vite qu'on ne croit)
- Préparer 3 questions que le jury pourrait poser
- Avoir une phrase de conclusion mémorisée pour finir proprement

## 5. Gestion du stress et du temps le jour J

### Avant l'épreuve

- Dormir (le cerveau consolide la mémoire pendant le sommeil)
- Manger un vrai repas (pas de jeûne, pas de surcharge)
- Arriver en avance pour s'installer calmement
- Relire ses fiches la veille, pas le matin même (pour les anxieux)

### Pendant l'épreuve

**Les 5 premières minutes :**
1. Lire TOUT le sujet / les consignes
2. Respirer 3 fois profondément
3. Identifier ce qu'on sait faire et ce qui va être difficile
4. Commencer par ce qu'on maîtrise (confiance + points assurés)

**Gestion du temps :**
- Écrire les horaires de fin par exercice/partie en haut de la copie
- Si on bloque : passer à la suite, y revenir après
- Garder 10 min de relecture INCOMPRESSIBLES

**Si panique :**
- Poser le stylo
- Respirer : inspirer 4 sec, bloquer 4 sec, expirer 4 sec
- Se rappeler : le jury ne cherche pas la perfection, il cherche la méthode et la réflexion
- Écrire quelque chose vaut toujours mieux que laisser blanc
