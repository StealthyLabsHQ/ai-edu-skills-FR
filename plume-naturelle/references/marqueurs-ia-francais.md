# Liste noire transversale des marqueurs IA en français

> **Quand charger ce fichier :** lorsque le texte à humaniser ne relève pas d'une discipline
> précise, ou en complément du fichier `patterns-par-discipline.md` pour un balayage complet.
> Cette liste regroupe les marqueurs IA les plus fréquents en français, toutes disciplines
> confondues.

---

## 1. Mots et expressions bannis (suppression ou remplacement obligatoire)

Ces mots/expressions sont des signaux IA quasi-certains quand ils apparaissent en densité.
Un ou deux dans un texte de 5 pages, ça passe. Trois ou plus, le texte est flagué.

### Catégorie A : Mots-valises et buzzwords

| Expression IA | Pourquoi c'est un signal | Remplacement |
|---|---|---|
| crucial | Superlatif favori des LLM, surutilisé x10 vs humain | important, clé, décisif (si vraiment décisif), central |
| fondamental | Idem, souvent non justifié | de base, premier, essentiel (si vraiment essentiel) |
| essentiel (en ouverture) | L'IA ouvre 1 phrase sur 5 avec ce mot | Reformuler sans ce mot |
| primordial | Variante de crucial/essentiel, même problème | nécessaire, indispensable |
| incontournable | Tout est "incontournable" pour l'IA | important, qu'on ne peut pas ignorer |
| indéniablement | Modalisateur fort que les humains utilisent rarement | clairement, c'est un fait, de toute évidence |
| intrinsèquement | Vocabulaire philosophique plaqué hors contexte | en soi, par nature |
| catalyseur | Métaphore chimique surutilisée | déclencheur, ce qui a lancé, moteur |
| synergie | Consultant-speak pur | collaboration, complémentarité, travail commun |
| paradigme (hors philo) | Jargon académique mal utilisé | modèle, cadre, logique |
| holistique | Fourre-tout new age | global, d'ensemble, complet |
| écosystème (hors bio) | Métaphore creuse | environnement, réseau, milieu |
| résilience (hors psycho) | Mot-valise post-Covid | résistance, solidité, capacité à tenir |
| disruptif / disruption | Anglicisme tech plaqué partout | qui bouleverse, changement brutal, rupture |

### Catégorie B : Métaphores mortes et images usées

| Expression IA | Remplacement |
|---|---|
| dans un monde en constante évolution | Supprimer entièrement |
| paysage (en constante mutation) | contexte, situation, environnement |
| tapisserie (au sens figuré) | ensemble, mosaïque |
| naviguer dans (les complexités) | gérer, composer avec, se débrouiller avec |
| orchestrer | organiser, coordonner |
| plonger dans | examiner, étudier, regarder de près |
| tisser des liens | créer des relations, travailler ensemble |
| au coeur de | au centre de, dans, ∅ (souvent supprimable) |
| pierre angulaire | base, fondation, point de départ |
| fer de lance | en tête de, qui mène |
| épine dorsale | structure principale, colonne vertébrale (si anatomie), sinon "base" |
| à l'aube de | au début de, au moment de |
| à l'ère de | aujourd'hui, maintenant que, depuis que |
| lever le voile sur | montrer, révéler, expliquer |
| ouvrir la voie | permettre, rendre possible |

### Catégorie C : Formules de remplissage

| Expression IA | Remplacement |
|---|---|
| il convient de noter que | ∅ (dire directement) |
| force est de constater que | on voit bien que, le fait est que |
| il est important de souligner que | ∅ (souligner directement) |
| il est intéressant de constater que | ∅ (constater directement) |
| il va sans dire que | ∅ |
| il est primordial de | il faut |
| en fin de compte | au final, bref |
| en définitive | au final, pour résumer |
| à cet égard | sur ce point, ∅ |
| dans cette optique | donc, alors, ∅ |
| eu égard à | vu, étant donné |
| en ce qui concerne | pour, sur |
| il est à noter que | ∅ |
| on peut observer que | ∅ |
| cela met en évidence le fait que | cela montre que |

### Catégorie D : Connecteurs mécaniques

| Connecteur IA | Quand le remplacer | Alternative |
|---|---|---|
| De plus | Quand il ouvre une phrase | ∅, Et, Aussi, Autre chose : |
| En outre | Toujours | ∅, Par-dessus le marché (si registre OK) |
| Par ailleurs | Quand il sert juste de transition | ∅, Sur un autre plan |
| Néanmoins | Quand il y en a plus d'un par page | Mais, Sauf que, Reste que |
| Toutefois | Idem | Or, Sauf que, Le problème c'est que |
| En revanche | Acceptable mais pas plus d'1 par page | Mais, À l'inverse |
| Cependant | Signal IA quand répété | Mais, Or, Pourtant |
| Ainsi | Quand il ouvre mécaniquement une conclusion | Donc, Du coup (si registre OK), Résultat : |
| En effet | Plus d'1 par page = signal | Car, Parce que, C'est que |
| À ce titre | Signal fort de style IA | ∅, Pour cette raison |

**Règle clé :** un paragraphe de 4 phrases peut ne contenir aucun connecteur.
La logique du propos fait le lien. Les connecteurs ne sont pas obligatoires.

### Catégorie E : Verbes de substitution à "être"

L'IA fuit le verbe "être" en le remplaçant par des verbes plus "soutenus".

| Verbe IA | Quand c'est un signal | Remplacement |
|---|---|---|
| constitue | "X constitue un Y" = signal quasi-certain | est |
| représente | "X représente un Y" = idem | est |
| incarne | Hors contexte littéraire/théâtral | est, montre, illustre |
| s'avère être | Pléonasme (s'avérer = se révéler vrai) | est, se révèle |
| s'inscrit dans | "Cette démarche s'inscrit dans..." | fait partie de, relève de |

## 2. Structures syntaxiques signalées par les détecteurs

### 2.1 La triade systématique

L'IA énumère presque toujours par trois. Deux ou quatre, jamais.

**Signal :** « la productivité, la qualité et la satisfaction client »
**Correction :** développer chaque point, ou énumérer par deux, ou par quatre.

### 2.2 Le parallélisme négatif

**Signal :** « Il ne s'agit pas seulement de X, mais aussi/surtout de Y »
**Correction :** dire directement ce qu'on veut dire.

### 2.3 La phrase d'ouverture universelle

**Signal :** « Dans un monde en constante évolution... », « De nos jours... »,
« Depuis toujours... », « À l'heure de la mondialisation... »
**Correction :** commencer par le concret, le terrain, le cas.

### 2.4 L'accumulation de participes présents

**Signal :** « ...permettant de... assurant... favorisant... »
**Correction :** remplacer chaque -ant par un mécanisme concret avec sujet-verbe.

### 2.5 La conclusion optimiste vague

**Signal :** « Les perspectives sont prometteuses », « L'avenir s'annonce... »
**Correction :** conclure sur du concret (que reste-t-il à faire ? qu'est-ce qui n'a pas marché ?).

## 3. Marqueurs de pronom et de ton

### 3.1 Abus du pronom "cela"

Les LLM utilisent "cela" 3 à 4 fois plus que la moyenne humaine écrite. C'est un signal
statistique fort que les détecteurs mesurent.

**Avant :**
> Cela montre que la digitalisation a un impact. Cela confirme les conclusions de
> l'étude précédente. Cela soulève également des questions.

**Après :**
> La digitalisation a un impact mesurable. L'étude précédente arrivait à la même
> conclusion. Reste une question ouverte.

**Règle :** maximum 2 "cela" par page. Remplacer par le sujet réel ou reformuler.

### 3.2 Absence de "je" (quand le format l'autorise)

Dans un mémoire professionnel, un rapport de stage, une lettre de motivation, l'absence
de "je" est un signal IA. L'IA produit du texte impersonnel par défaut.

### 3.3 Ton excessivement neutre

Un vrai auteur a un point de vue. Il montre une légère préférence, un scepticisme assumé,
une surprise. L'IA reste neutre et équidistante. Les détecteurs mesurent cette neutralité
constante via l'analyse de sentiment.

**Avant (neutre IA) :**
> L'outil présente des avantages et des inconvénients.

**Après (point de vue humain) :**
> Pennylane fait bien la collecte, mais son module de révision est frustrant.

## 4. Signaux statistiques invisibles

Ces signaux ne se voient pas à l'oeil nu mais les détecteurs LLM les mesurent.

### 4.1 Perplexité basse

Le texte est composé de mots "évidents" à chaque position. Chaque mot est celui qu'un
LLM aurait prédit. Solution : mots précis, noms propres, chiffres, tournures non standard.

### 4.2 Burstiness faible

Toutes les phrases font la même longueur et la même complexité. Solution : alterner
entre phrases de 3-5 mots et phrases de 30+ mots. Viser un écart-type > 8 mots.

### 4.3 Ratio type/token bas

Le texte recycle les mêmes mots. Solution : vocabulaire technique de terrain, jargon
métier, hapax (mots qui n'apparaissent qu'une fois).

### 4.4 Entropie constante

Le texte maintient une densité informationnelle uniforme. Solution : alterner passages
denses (analyse chiffrée) et passages légers (anecdote, aparté, question).

### 4.5 Distribution de tokens prédictible

À chaque position, le mot est celui du top-3 du LLM. Solution : varier les débuts de
phrases, placer l'information inattendue en position forte.

## 5. Checklist rapide — 15 signaux à scanner

```
[ ] "Crucial" / "fondamental" / "essentiel" en ouverture : 0 occurrence
[ ] "De plus" / "En outre" / "Par ailleurs" : max 2 dans tout le texte
[ ] "Il convient de" / "Force est de constater" : 0
[ ] "Dans un monde en constante évolution" : 0 (éliminatoire)
[ ] "Cela" : max 2 par page
[ ] Triades (X, Y et Z) : max 2 par page
[ ] Tirets cadratins : 0 idéalement, max 2 par page
[ ] Toutes les phrases entre 15-20 mots (pas de variation) : NON
[ ] Tous les paragraphes de même longueur (±20%) : NON
[ ] Aucun exemple avec nom/chiffre/lieu précis : NON
[ ] Aucun "je" dans un format qui l'autorise : NON
[ ] Registre uniformément soutenu sans variation : NON
[ ] Chaque étape du raisonnement explicitement reliée : NON
[ ] Exemples hypothétiques ("Imaginons...") : 0
[ ] Conclusion optimiste vague : NON
```

Si plus de 5 cases sont cochées "signal", le texte a besoin d'une réécriture profonde.

## 6. Comment utiliser ce fichier

1. **Scanner le texte** avec la checklist rapide (section 5).
2. **Identifier les catégories dominantes** (A-E en section 1) et remplacer.
3. **Vérifier les structures syntaxiques** (section 2).
4. **Contrôler le ton et les pronoms** (section 3).
5. **Croiser avec les signaux statistiques** (section 4) pour les cas difficiles.
6. **Si le texte est d'une discipline précise**, compléter avec `patterns-par-discipline.md`.
